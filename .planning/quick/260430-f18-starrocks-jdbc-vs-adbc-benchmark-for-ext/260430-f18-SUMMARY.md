# StarRocks JDBC vs ADBC Benchmark CLI for External StarRocks

One-liner: Reproducible TPC-H benchmark CLI comparing JDBC (MySQL protocol, port 9030) vs ADBC (Arrow Flight, port 9408) performance against the external StarRocks instance (sr-external) with ASCII table output.

## Summary

Created `benchmark/starrocks-jdbc-vs-adbc.py` — a CLI benchmark that mirrors the existing `benchmark/mysql-jdbc-vs-adbc.py` structure but targets the external StarRocks instance (sr-external) instead of the MySQL backend. It creates separate JDBC and ADBC catalogs, runs TPC-H queries through both paths with EXPLAIN ANALYZE, and prints a comparison table with per-query and aggregate (AVG/GEOM) metrics.
