- AVG = arithmetic mean of ratios across queries (skipping N/A rows). GEOM = `statistics.geometric_mean` (D-10).
- AVG/GEOM rows have the time columns (JDBC total, ADBC total, JDBC scan, ADBC scan) populated with mean of the corresponding column for non-N/A queries — these are informational. The summary semantics ARE the ratio columns.
- Row count mismatches go to stderr ONLY (D-11) — the table never shows row counts.

```python
def render_table(rows: list[dict], cols: list[tuple[str, int]]) -> str:
    sep = "+".join("-" * (w + 2) for _, w in cols)
    sep = f"+{sep}+"
    out = [sep]
    out.append("| " + " | ".join(f"{lbl:<{w}}" for lbl, w in cols) + " |")
    out.append(sep)
    for r in rows:
        out.append("| " + " | ".join(f"{r[lbl]!s:<{w}}" for lbl, w in cols) + " |")
    out.append(sep)
    return "\n".join(out)
```

<benchmark_loop_design>
<!-- D-18, D-20: warm-up pass + 3 measurement runs. RESEARCH.md §6 for ordering. -->

```
1. setup:
     - drop_catalog("bench_jdbc")  (defensive — recover from prior dirty state)
     - drop_catalog("bench_adbc")
     - create_jdbc_catalog(... bench_jdbc ...)
     - create_adbc_catalog(... bench_adbc ...)

2. discover queries:
     - glob queries/mysql/03-q*.sql, sort by query number ascending
     - filter by --queries flag if provided
     - read sql, strip comments, honor -- Skip: directive

3. warm-up phase (one pass per catalog, RESEARCH.md §6 recommends per-catalog grouping):
     - for each query: EXPLAIN ANALYZE on bench_jdbc, then on bench_adbc
       (results discarded — purpose is connection pool / buffer warm)

4. measurement phase (interleaved per query, RESEARCH.md §6):
     - for each query Q:
         for run in range(args.runs):    # default 3
             jdbc_plan_text = run_explain_analyze(bench_jdbc, Q)  # with timeout hint
             adbc_plan_text = run_explain_analyze(bench_adbc, Q)
             jdbc_total[Q].append(parse_summary_total(jdbc_plan_text))
             adbc_total[Q].append(parse_summary_total(adbc_plan_text))
             jdbc_scans[Q].append(parse_scan_nodes(jdbc_plan_text))
             adbc_scans[Q].append(parse_scan_nodes(adbc_plan_text))

         # compare row counts (one run is enough — must be deterministic)
         jdbc_count = SELECT count(*) on jdbc result
         adbc_count = SELECT count(*) on adbc result
         if jdbc_count != adbc_count: log to stderr (D-11, D-27)

5. aggregation:
     - per-query JDBC total avg = mean(jdbc_total[Q])
     - per-query ADBC total avg = mean(adbc_total[Q])
     - total ratio = jdbc_avg / adbc_avg (or "N/A" if either is timeout)
     - per-query scan-node ratios: for each operator_id present in BOTH catalogs'
       scan dicts, compute jdbc_total / adbc_total per run, then average per-id
       across runs, then average across ids → "scan ratio" for that query (D-09)
     - summary row AVG: arithmetic mean of total_ratios + scan_ratios
       across all non-N/A queries
     - summary row GEOM: statistics.geometric_mean(non_na_ratios)

6. teardown (always runs — try/finally + KeyboardInterrupt arm):
     - drop_catalog("bench_jdbc")
     - drop_catalog("bench_adbc")

7. render + print table
```

Timeout handling (D-19, D-25):
- Each EXPLAIN ANALYZE uses `with_timeout_hint(sql, args.timeout)` to inject SET_VAR.
- pymysql raises `pymysql.err.OperationalError` (or subclass) on timeout — catch it, mark that run as None, log "TIMEOUT" + query name to stderr, continue.
- If ANY run for a query times out, that query's ratio is "N/A" in the table (don't try to average partial data — keeps semantics simple).

FE SIGSEGV handling (CLAUDE.md "Pitfalls — StarRocks FE can SIGSEGV", RESEARCH.md §9):
- After catching `pymysql.err.OperationalError` ("Lost connection ..." or similar), execute a probe `SELECT 1` against StarRocks FE.
- If the probe fails too: print "✗ StarRocks FE appears to be down. Run `docker compose -f docker/docker-compose.yml restart sr-main` and retry." to stderr and exit 2.
- If the probe succeeds: continue (it was just a per-query timeout/error, not an FE crash).

```python
def discover_queries(queries_arg: str) -> list[tuple[int, pathlib.Path]]:
    """Return [(query_number, path), ...] sorted by query number ascending."""
    requested: set[int] | None
    if queries_arg.lower() == "all":
        requested = None
    else:
        try:
            requested = {int(x.strip()) for x in queries_arg.split(",") if x.strip()}
        except ValueError as e:
            print(f"✗ Invalid --queries value: {queries_arg!r} ({e})", file=sys.stderr)
            sys.exit(2)

    out: list[tuple[int, pathlib.Path]] = []
    for path in sorted(QUERIES_DIR.glob("03-q*.sql")):
        m = _QUERY_FILENAME_RE.match(path.name)
        if not m:
            continue
        qnum = int(m.group(1))
        if requested is not None and qnum not in requested:
            continue
        out.append((qnum, path))

    if not out:
        print(f"✗ No queries matched --queries={queries_arg!r}", file=sys.stderr)
        sys.exit(2)

    return out
```

```python
def run_explain_analyze(conn, sql: str, timeout_seconds: int) -> str:
    """Run 'EXPLAIN ANALYZE <sql with SET_VAR hint>' and return the plan text.
    Raises pymysql.err.OperationalError on timeout (D-19, D-25).
    """
    hinted = with_timeout_hint(sql, timeout_seconds)
    with conn.cursor() as cur:
        cur.execute("EXPLAIN ANALYZE " + hinted)
        rows = cur.fetchall()
        # EXPLAIN ANALYZE returns one column per row; concatenate.
        return "\n".join(row[0] for row in rows) if rows else ""
```

**12. Per-query result aggregator**:
```python
def aggregate_query_result(
    jdbc_total_runs: list[int | None],
    adbc_total_runs: list[int | None],
    jdbc_scans_runs: list[dict[int, int]],
    adbc_scans_runs: list[dict[int, int]],
) -> dict:
    """Compute per-query JDBC/ADBC total avg, total ratio, scan avg, scan ratio.

    Returns a dict ready for table rendering. Values may be None / 'N/A'.
    """
    # Total time averages — None means at least one run timed out
    jdbc_clean = [t for t in jdbc_total_runs if t is not None]
    adbc_clean = [t for t in adbc_total_runs if t is not None]
    jdbc_total_avg_ns = statistics.mean(jdbc_clean) if len(jdbc_clean) == len(jdbc_total_runs) else None
    adbc_total_avg_ns = statistics.mean(adbc_clean) if len(adbc_clean) == len(adbc_total_runs) else None
    total_ratio = (
        jdbc_total_avg_ns / adbc_total_avg_ns
        if jdbc_total_avg_ns is not None and adbc_total_avg_ns is not None and adbc_total_avg_ns > 0
        else None
    )

    # Scan ratios per D-09: average per-id ratio across runs, then average across ids
    # Operator IDs present in BOTH catalogs' scan dicts
    common_ids = (
        set.intersection(*(set(d) for d in jdbc_scans_runs))
        if jdbc_scans_runs and all(jdbc_scans_runs)
        else set()
    ) & (
        set.intersection(*(set(d) for d in adbc_scans_runs))
        if adbc_scans_runs and all(adbc_scans_runs)
        else set()
    )
    per_id_ratios: list[float] = []
    jdbc_scan_avg_ns_list: list[float] = []
    adbc_scan_avg_ns_list: list[float] = []
    for op_id in common_ids:
        jdbc_avg = statistics.mean(d[op_id] for d in jdbc_scans_runs)
        adbc_avg = statistics.mean(d[op_id] for d in adbc_scans_runs)
        jdbc_scan_avg_ns_list.append(jdbc_avg)
        adbc_scan_avg_ns_list.append(adbc_avg)
        if adbc_avg > 0:
            per_id_ratios.append(jdbc_avg / adbc_avg)
    scan_ratio = statistics.mean(per_id_ratios) if per_id_ratios else None
    jdbc_scan_avg_ns = statistics.mean(jdbc_scan_avg_ns_list) if jdbc_scan_avg_ns_list else None
    adbc_scan_avg_ns = statistics.mean(adbc_scan_avg_ns_list) if adbc_scan_avg_ns_list else None

    return {
        "jdbc_total_ms": jdbc_total_avg_ns / 1e6 if jdbc_total_avg_ns else None,
        "adbc_total_ms": adbc_total_avg_ns / 1e6 if adbc_total_avg_ns else None,
        "total_ratio": total_ratio,
        "jdbc_scan_ms": jdbc_scan_avg_ns / 1e6 if jdbc_scan_avg_ns else None,
        "adbc_scan_ms": adbc_scan_avg_ns / 1e6 if adbc_scan_avg_ns else None,
        "scan_ratio": scan_ratio,
    }
```

```python
def _fmt(v: float | None, decimals: int = 1) -> str:
    if v is None:
        return "N/A"
    return f"{v:.{decimals}f}"


def render_table(per_query_rows: dict[int, dict], summary: dict) -> str:
    """Render the wide ASCII table per D-05 with AVG/GEOM summary rows per D-10."""
    sep = "+" + "+".join("-" * (w + 2) for _, w in _COLS) + "+"

    def _format_data_row(label: str, row: dict) -> str:
        cells = [
            f"{label:<{_COLS[0][1]}}",
            f"{_fmt(row['jdbc_total_ms']):>{_COLS[1][1]}}",
            f"{_fmt(row['adbc_total_ms']):>{_COLS[2][1]}}",
            f"{_fmt(row['total_ratio'], 2):>{_COLS[3][1]}}",
            f"{_fmt(row['jdbc_scan_ms']):>{_COLS[4][1]}}",
            f"{_fmt(row['adbc_scan_ms']):>{_COLS[5][1]}}",
            f"{_fmt(row['scan_ratio'], 2):>{_COLS[6][1]}}",
        ]
        return "| " + " | ".join(cells) + " |"

    lines = [sep]
    header = "| " + " | ".join(f"{lbl:<{w}}" for lbl, w in _COLS) + " |"
    lines.append(header)
    lines.append(sep)
    for qnum in sorted(per_query_rows.keys()):
        lines.append(_format_data_row(f"Q{qnum:02d}", per_query_rows[qnum]))
    lines.append(sep)
    lines.append(_format_data_row("AVG", summary["avg"]))
    lines.append(_format_data_row("GEOM", summary["geom"]))
    lines.append(sep)
    return "\n".join(lines)
```

**14. Summary aggregation (AVG and GEOMEAN across queries)**:
```python
def compute_summary(per_query_rows: dict[int, dict]) -> dict:
    """Compute AVG (arithmetic mean) and GEOM (geometric mean) summary rows."""
    def _collect(key: str) -> list[float]:
        return [r[key] for r in per_query_rows.values() if r.get(key) is not None]

    def _agg(fn, key: str) -> float | None:
        vals = _collect(key)
        if not vals:
            return None
        try:
            return fn(vals)
        except statistics.StatisticsError:
            return None

    avg = {
        "jdbc_total_ms": _agg(statistics.mean, "jdbc_total_ms"),
        "adbc_total_ms": _agg(statistics.mean, "adbc_total_ms"),
        "total_ratio":   _agg(statistics.mean, "total_ratio"),
        "jdbc_scan_ms":  _agg(statistics.mean, "jdbc_scan_ms"),
        "adbc_scan_ms":  _agg(statistics.mean, "adbc_scan_ms"),
        "scan_ratio":    _agg(statistics.mean, "scan_ratio"),
    }
    geom = {
        "jdbc_total_ms": _agg(statistics.geometric_mean, "jdbc_total_ms"),
        "adbc_total_ms": _agg(statistics.geometric_mean, "adbc_total_ms"),
        "total_ratio":   _agg(statistics.geometric_mean, "total_ratio"),
        "jdbc_scan_ms":  _agg(statistics.geometric_mean, "jdbc_scan_ms"),
        "adbc_scan_ms":  _agg(statistics.geometric_mean, "adbc_scan_ms"),
        "scan_ratio":    _agg(statistics.geometric_mean, "scan_ratio"),
    }
    return {"avg": avg, "geom": geom}
```
